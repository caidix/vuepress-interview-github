# function

It might be useful to define your own function to control the behavior of what you want to externalize from webpack. webpack-node-externals (opens new window), for example, excludes all modules from the node_modules directory and provides some options to, for example, whitelist packages.

It basically comes down to this:

externals: [
  function(context, request, callback) {
    if (/^yourregex$/.test(request)){
      return callback(null, 'commonjs ' + request);
    }
    callback();
  }
],

# regex

Every dependency that matches the given regular expression will be excluded from the output bundles.

externals: /^(jquery|\$)$/i
1

In this case any dependency named jQuery, capitalized or not, or $ would be externalized.
